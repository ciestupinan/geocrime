GeoCrime

This project displays up-to-date data from the San Francisco Police Department police reports. Reports included are those for incidents that occurred starting January 1, 2018 onward and have been approved by a supervising officer.

Incidents can be filtered by date, time of day, category type, and resolution. Each category has a color code. To learn more about what types of reports are filed as those categories, hover over the color code to read the tooltip.

To display incidents at the same location, I used the Overlapping Marker Spiderfier library. When one of these incidents are clicked, all of the incidents at that location with spider out.
